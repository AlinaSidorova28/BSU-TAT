### Test-case 1. Регистрация / авторизация через google аккаунт  
**Предусловия**: пользователь не авторизован в системе и не зарегистрирован, находится на главной (или любой другой) странице сайта.  
**Шаги**:  
  1. Выбирает в правом верхнем углу иконку "человечек".
  2. Во всплывшем окне входа Выбирает среди иконок "Войти с помощью своего профиля" иконку google.  
  3. Во всплывшем окне выбирает свой аккаунт.  

**Ожидаемый результат:** сайт открыл страницу личного кабинета пользователя с заполненными полями "Имя", "Фамилия" и "Ваш email" (из google аккаунта), в верхнем правом углу рядом с иконкой "человечка" написано имя авторизовавшегося пользователя.

### Test-case 2. Фильтры для поиска (цвет, размер)  
**Предусловия**: пользователь находится на странице с каталогом определенных вещей, например, "Женские майки, топы". На странице находится список фильтров.  
**Шаги**:  
  1. Пользователь сортирует товары по цвету, нажав кнопку "цвет" и выбрав "синий".  
  2. Пользователь сортирует товары по размеру, нажав кнопку "размер" и выбрав "S".

**Ожидаемый результат:** получен список товаров, где каждый товар обладает свойством цвета "синий", которое может быть найдено в описании товара в результатах поиска (при клике на "предпросмотр") в виде конструкции "Цвет:" + название цвета "синий" + круг цвета "синий" под указанием цены. Также у каждого товара в таблице "Размер" размер S доступен для выбора, т.е не перечеркнут.

### Test-case 3. Сортировка по цене  
**Предусловия**: пользователь находится на странице с каталогом определенных вещей, например, "Женские майки, топы". На странице находится список фильтров.  
**Шаги**:  
  1. Пользователь нажимает на кнопку "Сортировка" и выбирает "Цена по возрастанию".  

**Ожидаемый результат:** получен список товаров, где все товары расположены в порядке неубывания цены с учетом скидок, т.е цена каждого следующего товара больше или равна цене предыдущего.

### Test-case 4. Добавление товара в корзину  
**Предусловия**: пользователь находится на странице описания конкретного товара, например "Джемпер женский", Модель: 112456, корзина товаров пуста.  
**Шаги**:  
  1. Выбирает количество товара, которое желает добавить в корзину (1), размер (XS) и цвет ("хаки").
  2. Нажимает на кнопку "В корзину".  

**Ожидаемый результат:** всплывает окно "Корзина" с информацией о только что добавленном товаре, ("Джемпер женский", Цвет "хаки", Артикул 3104-20, Сумма: 4.99 BYN, Количество 1), указанием общей стоимости заказа "Заказ на сумму: 4.99 BYN" и кнопкой "Перейти в корзину". В корзине появился товар "Джемпер женский" с параметрами: Цвет "хаки", Модель: 112456, Размер: "XS", Количество 1, Сумма 4.99 BYN, Общая стоимость заказа 4.99 BYN.

### Test-case 7. Изменение пароля от аккаунта  
**Предусловия**: пользователь находится в личном кабинете, внизу страницы есть две формы ("Введите новый пароль", "Повторите новый пароль") под фразой "Изменить пароль".  
**Шаги**:  
  1. Вводит в поле "Введите новый пароль" новый пароль для аккаунта, состоящий не менее, чем из 6 знаков.
  2. Вводит тот же пароль в поле "Повторите новый пароль".  
  3. Нажимает клавишу Enter или кнопку "Сохранить".  

**Ожидаемый результат:** вверху страницы появляется зеленая надпись "Изменения сохранены", новый пароль привязывается к аккаунту сайта markformelle.

### Test-case 8. Попытка изменить пароль невалидными данными  
**Предусловия**: пользователь находится в личном кабинете, внизу страницы есть две формы ("Введите новый пароль", "Повторите новый пароль") под фразой "Изменить пароль".  
**Шаги**:  
  1. Вводит в поле "Введите новый пароль" новый пароль для аккаунта, состоящий менее, чем из 6 знаков. / Вводит в поле "Повторите новый пароль" строку, не совпадающую со строкой из поля "Введите новый пароль".  
  2. Нажимает клавишу Enter или кнопку "Сохранить".  

**Ожидаемый результат:** вверху страницы появляется красная надпись "Пароль должен быть не менее 6 символов длиной" / "Неверное подтверждение пароля", пароль не сохраняется.

### Test-case 9. Добавление товара в "Избранное"  
**Предусловия**: пользователь находится на странице описания конкретного товара, например, "Топ женский", Модель: 112527, у пользователя нет товаров в "Избранном".  
**Шаги**:  
  1. Выбирает цвет товара, который хочет добавить в "Избранное", например "коралловый +печать".
  2. Нажимает на кнопку с сердечком справа от кнопки "Добавить в корзину".  

**Ожидаемый результат:** в правом верхнем углу на вкладке избранного (иконка "сердечко") появляется цифра 1, обозначающая количество товаров в "Избранном". Во вкладке "Избранное" появляется выбранный товар "Топ женский", Артикул 19-5871-5, Цвет "коралловый +печать" с кнопкой "Добавить к заказу?".
